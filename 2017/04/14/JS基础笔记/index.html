JS基础笔记

发表于 2017-04-14 | 分类于 web前端 | 阅读次数 |

typeof

typeof()可以返回对象的类型，目前有以下几种类型：

  • number 数字
  • string 字符串
  • boolean 布尔型
  • undefined 未定义
  • function 函数
  • object 对象

    alert()函数

    通过调用alert()函数，可以在页面中弹出一个新的提示窗口，内容由传入的参数决定。通过alert()函数主要可以用来进行逐步测试，通过插入代码的不同位置来判断代码错误的地方。初学者最好每写一段就测试一下。

    变量作用域

    变量有两种，一种局部变量，定义在函数中只在本函数中使用；另一种全局变量，不定义在任何一个函数里，任何地方都可以使用。在工具中，尽量少使用全局变量。

    命名规范(匈牙利命名法)

类型 类型 前缀
数组 Array a
布尔值 Boolean b
浮点数 Float f
函数 Function f

对象 Object o
正则表达式 RegExp r

变体变量 Variant v

单词首字母大写，例如，oDivUserLogin.

运算符

算术

“+ - * / %”不同的运算符不止能计算，还能通过计算达到一些意想不到的效果。

  • 隔行变色，通过对元素ID的取模%计算，可以实现隔行变色的功能。
  • 倒计时功能，n秒倒计时，转换为 parseInt(n/60) 分 n%60 秒.

    赋值与比较

    赋值：= += -= *= /= %=
    比较：< > >= <= == === != !==

逻辑与运算优先

逻辑：&& 且、|| 或、 ！否
运算优先级：括号

程序流程控制

判断： if, switch, ?:(三目运算符if else 的简写)
循环： while, for
跳出： break(终止整个循环), continue(终止本次循环，进入下次循环)

数组

可变参：arguments;
currentStyle, Style.语法只能获取行间元素，而currentStyle还可以获取除行间元素以外的属性。(获取计算后的样式 或者 当前样式 或者 最终样式)不支持火狐，支持IE，火狐下使用getComputedStyle.

数组排序

使用sort()函数能进行字符串排序，但是直接进行数字排序会出错，这时可以在sort()传入一个比较函数作为参数。

数组连接concat

数组分隔符join

arr.join(':'); //连接符可以自定义，此处定为:

数组的插入与删除 splice

splice(); //开始位置之后， 删除长度 ，插入内容

splice先从数组中删除元素，再加入新的元素.
