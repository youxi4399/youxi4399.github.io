IE6下双边距BUG

IE6,7下具有双边距BUGU，其中具体表现为：

  • 块元素有浮动和横向的margin，而横向的margin会被放大成2倍。解决办法：使块元素变为内嵌元素。
  • 需要元素浮动时，在同一个父级元素下都需要设置浮动，比如一个向左浮动，一个向右浮动。
  • li下几px的间隙问题,li本身没有浮动但内容浮动后li下就会多出来几px，解决办法vertical-align(垂直对齐); li加浮动。
  • IE下最小高度问题，最小为19px，不足19px的按19计算，仅限于块级元素。解决办法：font-size:0pxl; overfllow:hidden.

    定位position标签

    相对定位

    position: relative;
  1. 不影响元素本身特性
  2. 不使元素脱离文档流；
  3. 如果没有定位偏移量，对元素本身没有任何影响。

    绝对定位

    position: absolute;
  4. 脱离文档流
  5. 是内嵌元素支持宽高
  6. 块属性标签内容撑开宽度
  7. 如果有定位父级则定位于父级发生偏移，没有则相对于文档发生偏移。

    定位层级z-index

    定位元素默认后者层级高于前者；在使用时不需要挨个设置，只需要把想要提前的设置为1就好。

    CSS3 2D转换

    CSS3中2D转换使用transform属性。其中有很多属性值：
  • translate() 平移
  • rotate() 绕几何中心进行旋转
  • skew() 分别给出x轴和y轴的角度值，然后图形会在x轴参数时以y轴为起点变形；y轴参数时以x轴为起点变形。
  • scale() 按照倍数进行x, y 上的变化即宽度与高度的变化。大于1的数进行放大，小于1的数进行缩小。
  • matrix() 需要6个参数，包含数学函数，允许你：旋转、缩放移动以及倾斜元素。skew, scale, rotate, translate
