JavaScript属性修改

属性读操作：获取找到：元素.属性名
属性写操作：修改，删除，添加。
通过”.”操作符可以得到属性值，再使用类似于赋值的方法，可以对其重新的修改。如果要读取标签内的内容，则需要用到innerHTML属性。

添加内容

通过使新的内容包含原来的内容加到新内容中。例如：
innerHTML = innerHTML + oText.value;
等同于
innerHTML += oText.value;

属性读写操作注意事项

写入”script”标签时应该注意使用window.onload()使其在页面加载完成后才使用script脚本。
使用JS时，-是不被允许的。如font-size -> fontSize；去掉-改为后面的第一个字母大写。

设置class属性

因为在JavaScript中class是一个保留字，所以如果要设置class属性的话(JavaScript关键字：var function 等；保留字:class 等)就需要设置为className;如果需要设置的属性太多，可以先设置一个class类，再在JavaScript中调用class。

相对路径不要拿来做判断

  1. 在编写html文件时，不要用相对路径作为判断依据，非常得不方便，在兼容性上容易出问题，另外也不通用。
  2. 颜色值也不要拿来做判断，因为颜色值得表达方式有很多种，比如 red #f00 rgb(255, 0, 0);
  3. innerHTML也不要拿来做判断。

    注意IE的兼容性

    IE有3个老版本浏览器，IE6, IE7， IE8。在三个浏览器中很多标签或者表达方式都是不能正常解析出来的，需要考虑兼容性。比如设置Div.style.float的属性值，IE(styleFloat), 非IE(cssFloat)绕过float，使用class(className)来绕过浮动。

    .操作符

    .操作符在多次使用的时候不方便，这时可以使用[]来获取；比如：oDiv.style.oAttr.value -> oDiv.style[oAttr.value]
    因为.操作符后面的值无法修改，而[]里的值可以随便写，在[]中最好有引号。
